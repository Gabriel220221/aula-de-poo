# Projeto Garrafa

Este projeto representa uma garrafa no sistema.

A classe Garrafa possui:

Atributos:
- cordatampa
- marca
- materialdagarrafa

Métodos:
- mudarCorTampa() -> muda a cor da tampa
- mudarMarca() -> muda a marca da garrafa

Os métodos possuem validação com if para não aceitar valores nulos.
